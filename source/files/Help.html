MyWiki Syntax

Headings

H1: =heading text
H2: ==heading text
H3: ===heading text

The first '=' character must be the first character on the line.

Horizontal Rule

----

A line consisting of four or more dashes and no other characters.

Bold Text

''bold text''

Underlined Text

__underlined text__

Italic Text

~~italic text~~

Ordered List

# list item (starting at the beginning of a line)

Unordered List

* list item (starting at the beginning of a line)

Hyperlink

[[href]]
[[href|anchor text]]
Note: href must start with:

  • http://
  • https://
  • file://
  • mailto://

Any href that does not start with one of the above is treated as a search within the deck.

For special characters to be found in searches, they must be escaped in the anchor text. e.g., \$

Paragraphs

Paragraphs are defined by two or more newlines in succession.

The left margin is set by the first characters of the paragraph:

  • Two colons: 60 pixels
  • One colon: 30 pixels
  • Any other character: 0 pixels
